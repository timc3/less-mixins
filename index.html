    Welcome to Less Mixins.

    This is a collection of mixins that I use often with different projects. Each one tries to by cross-browser friendly.

    Usage

    Usage is actually rather simple, you need to include the contents of less_mixin.less in to your project.

    1. Download / Clone this project.
    2. Copy less_mixin.less into the same directory as the rest of your .less files
    3. @import "less_mixin"; near the head of you main .less file.

    4. When you wish to use one of the mixins just include it in the declaration of the CSS block:

      .roundedbox { border: 1px solid black; z-index: 999; .mixin-rounded(8px); }

    The above just added rounded corners with a radius of 8px to any element with the class roundedbox applied to it.
